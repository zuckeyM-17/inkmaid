description: TypeScript型エラーを検出し修正します
---

プロジェクト内のTypeScript型エラーを検出し、修正してください。

## 手順

1. **型エラーの検出**
   ```bash
   pnpm --filter web build
   ```
   または
   ```bash
   tsc --noEmit
   ```

2. **エラーの分類**
   - 型定義の欠落
   - 型の不一致
   - null/undefined関連
   - importエラー

3. **修正の実施**
   - 適切な型定義を追加
   - 型アサーションが必要な場合は理由をコメント
   - `any` は使用せず、適切な型を推論

4. **検証**
   - 修正後、再度ビルドして型エラーがないことを確認
   - 機能が正常に動作することを確認

各エラーについて、修正内容と理由を説明してください。
