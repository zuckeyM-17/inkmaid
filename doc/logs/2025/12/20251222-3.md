# 開発ログ 2025-12-22 #3

## 実装内容

### プロジェクト名変更機能

プロジェクト詳細ページでプロジェクト名をインライン編集できる機能を実装しました。

**API:**
- `diagram.renameProject` - プロジェクト名を変更するmutation

**UI:**
- ヘッダーのプロジェクト名をクリックするとインライン編集モードに
- Enterキーで確定、Escapeキーでキャンセル
- 編集アイコン（✎）をホバー時に表示
- 保存中はローディング表示

## 使い方

1. プロジェクト詳細ページ（`/projects/[id]`）を開く
2. ヘッダーのプロジェクト名をクリック
3. 新しい名前を入力
4. Enterキーまたは ✓ ボタンで保存
